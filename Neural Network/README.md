# Logistic Regression

### Requirements
- Python3
- PIP3 (Python Package Manager)

### How to Run
**1.** Clone this Repo <br />
**2.** Go into Neural Network folder <br />
**3.** Install Packages (pip3 install --upgrade -r requirements.txt) <br />
**4.** Run the project (python3 neural_network.py) <br />
